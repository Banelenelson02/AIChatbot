# 🤖 Java ChatBot Project

A simple Java-based terminal chatbot that responds to user commands.

## Features
- Tells time ⏰ (`time`)
- Shows date 📅 (`date`)
- Tells jokes 😂 (`joke`)
- Help menu 📋 (`help`)

## How to Run
```bash
# Compile
javac src/ChatBot.java -d bin/

# Run
java -cp bin/ ChatBot
